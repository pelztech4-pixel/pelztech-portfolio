# Professional Portfolio Website Template

This is a clean, responsive portfolio website template for a Full Stack Developer.

## How to Edit

Open these files in VS Code:

- `index.html` — change your name, job title, about text, projects, contact details, LinkedIn and GitHub links.
- `style.css` — change colors, spacing, and layout.
- `script.js` — controls the mobile menu and footer year.

## How to Preview Locally

Double-click `index.html` to open it in your browser.

## How to Upload to GitHub Pages

1. Create a GitHub account or login.
2. Create a new repository named `portfolio`.
3. Upload `index.html`, `style.css`, `script.js`, and `README.md`.
4. Go to repository `Settings`.
5. Click `Pages`.
6. Under `Branch`, choose `main` and `/root`.
7. Click `Save`.
8. Your live website will be available at:
   `https://yourgithubusername.github.io/portfolio/`

## Important

Replace all placeholders before applying for jobs.
